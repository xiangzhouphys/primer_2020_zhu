# primer_2020
primer_2020 is a prime number library. It provides functions
for generating primes and factorization.

## Installation

```
$ git clone https://github.com/MartinThoma/primer_2020.git
$ cd primer_2020
$ pip install -e .
```

## Usage

```
>>> import primer_2020
>>> primer_2020.factorize(28)
>>> [2, 2, 7]
```
